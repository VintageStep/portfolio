As I said earlier, this project was made using Flask, It's also a part of a web developer bootcamp over Udemy, more details here. And while the back-end part hasn't changed significantly, I just added more data, the front-end couldn't be more different.

I took the opportunity to own this project by creating a layout and style on my own and applying what I know about CSS. I found plenty of challenges, mostly related to responsiveness and making sure content doesn't overflow or affect its container in undesired ways. I'm sure the CSS style sheet is a bit messy, but I'm aware of possible improvements that can be made.

An interesting feature of the project is that most of the project is dynamic excluding the about page and the pages of the projects themselves, like this very one! This gives me the chance to add more projects to the main layout without effort while being able to personalize pages as much as I want, I will take advantage of this later, on the page for front-end snippets.

Lastly, I designed all the images used in this project using Figma. Some components of the wireframes come from here. And I have to thank Dave Whitley for them. As you might have guessed the wireframes on the portfolio are not the ones I originally used when designing the different projects, but I think they convey the idea much better.
